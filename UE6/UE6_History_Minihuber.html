Die Geschichte von HTML

Entstehung

In CERN entstand 1989 ein Projekt, für das strukturieren von Daten, die zwischen zwei Standorten ausgetauscht werden mussten. Es dauerte drei Jahre, bis am 3. November 1992 die erste Version von HTML erschien.

Versionen

Es gab schon ziemlich viele verschiedenen HTML-Versionen.
Hier ist eine Liste von ihnen:

  1. HTML (1992)
  2. HTML (1993)
  3. HTML+ (1993)
  4. HTML 2.0 (1995): Einführung der Formulartechnik
  5. HTML 3.0: nicht erschienen
  6. HTML 3.2 (1997): Einbindung von Tabellen, Applets und mehr
  7. HTML 4.0 (1997): Stylesheets (CSS) erscheinen auf der Bildfläche, dazu Skripten und Frames
  8. HTML 4.01 (1999): Korrigiertes Update von HTML 4.0
  9. XHTML 1.0 (2000): Neue Formulierung von HTML 4.01
  10. XHTML 1.1 (2001): Strikte Version in Folge der Aufteilung von XHTML in Module
  11. XHTML 2.0 (2006): während der Arbeit wegen dem künftigen HTML5 eingestellt
  12. HTML5 (2014): Neues Vokabular und das Fundament für den Standard bis heute, aufbauend auf XHTML
  13. HTML 5.1 (2016)
  14. HTML 5.2 (2017)
